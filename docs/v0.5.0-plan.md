wave are parallelizable.

## Risk Register

| Risk | Impact | Mitigation |
|------|--------|------------|
| `internal/jwt` correctness (RFC 7519 compliance without third-party lib) | High | Exhaustive test vectors from RFC; property-based tests on encode/decode roundtrip |
| `runtime.KeepAlive` dead-store elision edge cases | Medium | Test with `-gcflags=-m` to verify inlining; benchmark zeroing path |
| Conformance suite flakiness (live API calls) | Medium | Budget cap ($0.50/run), retry logic, nightly-only cadence |
| Prometheus metric cardinality drift | Medium | Static analysis of label values; test worst-case time series count |
| Coverage regression from new packages | Low | CI enforces 85% per-package; track during each wave |
